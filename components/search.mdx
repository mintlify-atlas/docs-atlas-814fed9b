## Search Flow

The search components work together in this flow:

1. User enters search query in **SearchForm**
2. Form submits and triggers parallel searches for tracks, albums, and artists
3. Results are stored in **useSearchStore**
4. Results are displayed in respective list components:
   - **TrackListSerach** for tracks
   - **AlbumsListSearch** for albums
   - **ArtistListSearch** for artists

### Complete Search Implementation

```tsx
import { SearchForm } from 'components/SearchComponents/components/SearchForm';
import { TrackListSerach } from 'components/SearchComponents/components/TrackListSearch';
import { AlbumsListSearch } from 'components/SearchComponents/components/AlbumsListSearch';
import { ArtistListSearch } from 'components/SearchComponents/components/ArtistListSearch';
import { useSearchStore } from 'store/searchStore/SearchStore';
import { View, StyleSheet } from 'react-native';
import { createMaterialTopTabNavigator } from '@react-navigation/material-top-tabs';

const Tab = createMaterialTopTabNavigator();

export function SearchScreen() {
  const { trackList, albumsList, artistList } = useSearchStore();

  return (
    <View style={styles.container}>
      <SearchForm />
      
      <Tab.Navigator>
        <Tab.Screen name="Tracks">
          {() => <TrackListSerach topSongs={trackList} />}
        </Tab.Screen>
        
        <Tab.Screen name="Albums">
          {() => <AlbumsListSearch albumArray={albumsList} />}
        </Tab.Screen>
        
        <Tab.Screen name="Artists">
          {() => <ArtistListSearch artistArray={artistList} />}
        </Tab.Screen>
      </Tab.Navigator>
    </View>
  );
}

const styles = StyleSheet.create({
  container: {
    flex: 1,
    backgroundColor: '#000'
  }
});
```

## State Management

The search components use:

- `useSearchStore()` - Stores search results for tracks, albums, and artists
- `useAlbumStore()` - Stores selected album for navigation
- `useArtistStore()` - Stores selected artist for navigation
- `useBottomSheetStore()` - Controls track options modal
- `useModalTrack()` - Manages track detail information

## Search Services

The SearchForm component uses these search functions:

```typescript
// Search tracks
const tracks = await serachTracksFuntion(query);

// Search artists
const artists = await serachArtistFuntion(query);

// Search albums
const albums = await searchAlbumsByAxios(query);
```

All three searches run in parallel using `Promise.all()` for optimal performance.
