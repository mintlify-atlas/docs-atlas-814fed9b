## What is HiBeat Pro?

HiBeat Pro is a feature-rich React Native music streaming application that brings the power of YouTube Music to your mobile device. Built with modern React Native technologies, it provides a seamless music listening experience with advanced features like background playback, offline downloads, and intelligent search.

<Note>
HiBeat Pro is designed as a learning project to demonstrate React Native CLI capabilities while providing a fully functional music streaming experience.
</Note>

## Technology Stack

HiBeat Pro is built with cutting-edge technologies:

- **React Native 0.74.3** - Cross-platform mobile development
- **TypeScript** - Type-safe development
- **YouTube Music API** (`ytmusic-api`) - Music data and streaming
- **React Native Track Player** - Advanced audio playback with background support
- **Zustand** - Lightweight state management
- **TanStack Query** - Efficient data fetching and caching
- **FFmpeg Kit** - Audio processing and metadata embedding

## Architecture Overview

The application follows a clean, modular architecture:

```
src/
├── components/      # Reusable UI components
├── screens/         # Screen-level components
├── services/        # Business logic (streaming, downloads, player)
├── hooks/           # Custom React hooks for data fetching
├── store/           # Zustand state management
├── navigation/      # React Navigation setup
└── interfaces/      # TypeScript type definitions
```

## Platform Support

<Info>
Currently, HiBeat Pro is **Android-only**. iOS support is not available in the current version.
</Info>

**Minimum Requirements:**
- Android SDK 24 (Android 7.0) or higher
- Node.js 18 or higher
- React Native 0.74.3
